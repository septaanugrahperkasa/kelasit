# Kelas IT

Kelas IT adalah sebuah proyek berbasis **Node.js dan Express.js** yang dikembangkan sebagai **media pembelajaran dan eksplorasi dasar pengembangan aplikasi web** menggunakan arsitektur server-side rendering (SSR) dengan **EJS**. Repository ini berfungsi sebagai **koleksi eksperimen, latihan, dan implementasi fitur dasar backend web**.

- Menjadi **arsip pembelajaran dan referensi pribadi** terkait pengembangan aplikasi web berbasis Express
